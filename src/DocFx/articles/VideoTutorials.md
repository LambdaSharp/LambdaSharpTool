# Video Tutorials

## Getting Started

It's easy to get started building Serverless .NET application on AWS with 位#. In in this 10 minute tutorial, we will install the 位# CLI, create a configuration for it, and a deployment tier for your 位# modules. Along the way, I will explain what resources are part of the configuration and deployment tier and what purpose they play.

## First Module

In this video, I'm going through the steps for creating your first Serverless .NET module for AWS using  位#. The whole tutorial takes less than 10 minutes. Along the way, I also show how to use CloudFormation parameters to parametrize your module and show how they are resolved interactively at deployment time.
